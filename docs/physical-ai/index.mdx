## About This Textbook

This textbook is designed for:
- **Students**: Learning about Physical AI and Humanoid Robotics
- **Educators**: Using it as a curriculum resource for robotics courses
- **Developers**: Implementing real-world robotics systems

### Learning Approach

This textbook follows a structured learning path:
1. **Foundations**: Understanding Physical AI and embodied intelligence
2. **ROS 2**: Building robotic systems with ROS 2 architecture
3. **Simulation**: Creating realistic environments with Gazebo and Unity
4. **AI Integration**: Advanced perception and manipulation with NVIDIA Isaac
5. **Advanced Systems**: Voice-to-action and cognitive planning
6. **Capstone**: Complete integration project

### Navigation

Use the sidebar to navigate through the 14 chapters, each building upon the previous ones. Each chapter includes:
- Clear learning outcomes
- Theoretical concepts
- Practical code examples
- Exercises and assessments
- Real-world use cases

### Technical Requirements

To follow along with the practical examples, you'll need:
- Ubuntu 22.04 LTS
- ROS 2 Humble Hawksbill
- Python 3.10+
- Node.js 18+ (for this documentation)
